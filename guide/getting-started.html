🚀 入门指南 ​

欢迎使用 My Docs 文档模板！本指南将帮助你快速搭建属于自己的文档站点。

前置要求 ​

  • Node.js 18+（推荐 20 LTS）
  • 包管理器（npm / yarn / pnpm）

快速开始 ​

1. 克隆模板 ​

bash
git clone https://github.com/aiastia123/my-docs.git
cd my-docs

2. 安装依赖 ​

bash
npm install

3. 启动开发服务器 ​

bash
npm run docs:dev

访问 http://localhost:5173 即可预览。

4. 构建生产版本 ​

bash
npm run docs:build

构建产物在 docs/.vitepress/dist 目录中。
